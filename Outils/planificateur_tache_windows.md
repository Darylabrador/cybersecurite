# Planificateur de tâche windows

## Pourquoi l'utiliser ? 

Permet de lancer l'exécution d'un programme ou d'une commande suivant un scheduler.

<br>

## Points positifs

- Lancer automatiquement un programme
- Lancer automatiquement une commande
